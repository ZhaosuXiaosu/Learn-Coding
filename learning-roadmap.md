# 编程语言学习路线 - 详细时间线拆解

## 总体学习顺序

```
HTML/CSS → Python → Lua → Java → C → C++
```

**总预期时间**: 6-9 个月（每天 2-3 小时）

## 时间调整建议

| 情况 | 调整方案 |
|------|---------|
| 有编程基础 | 可压缩 Python/Lua 至 4 周 |
| 时间充裕 | 每阶段增加 2 周做更深入项目 |
| 时间紧张 | 优先 Python → Java → C（跳过 Lua） |
| 目标导向 | 根据目标调整顺序（Web: HTML/CSS/Java; 游戏：Lua/C++） |

---

## 关键学习原则

1. **不要同时学多门语言** — 精通一门再学下一门
2. **做项目驱动学习** — 每门语言至少完成 2-3 个项目
3. **理解而非死记** — 关注"为什么这样设计"
4. **定期复习** — 学新语言时回顾旧知识
